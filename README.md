## How to Run:
 - Ensure that `minesweeper.py` and `run_game.py` are in the same directory.
 - In the terminal, execute the `run_game.py` script:    
    ```
    python run_game.py
    ```
## Requirements
Before running the game, install the required Python libraries. You can install all necessary dependencies using `pip`:
```
pip install numpy gymnasium pygame six
```

Alternatively, execute the `setup.py` folder by running this:
```
pip install -e .
```
